# phreda


## Overview

The R pacakge **phreda** is a fork of Wenjie Wang's **reda** package, and  
provides functions for

- simulating phenological event data using (slightly hijacked) utilities from
reda.


## Installation

```R
devtools::install_github("lzachmann/phreda")
```
